Sublime text 3 install and settings on WIN10.

2021-11-12 Sublime

1. Install Softwares

  • Sublime text 3
  • Texlive
  • SumatraPDF

Add paths to system environment variables, including

E:\Sublime3\
E:\Sublime3\SumatraPDF\
E:\TexLive\2019\

2. Sublime Settings

1. Package Control

  • ctrl + shift + p
  • Type Install Package Control, press enter

2. Latextools - latex

  • ctrl + shift + p
  • Type Package Control: Install Package, press enter
  • Type Latextools, press enter
  • Find LaTeXTools.sublime-settings under Latextools folder
  •   "texpath" : "C:\\texlive\\2019\\bin\\win32;$PATH",
  "distro"  : "texlive",
  "sumatra" : "E:\\Sublime3\\SumatraPDF\\SumatraPDF.exe",
  "builder" : "simple",
  "builder_settings" : 
      "windows" : {
          // See README or third-party documentation
          "script_commands":[
              "pdflatex -synctex=1 -interaction=nonstopmode",
              "bibtex",
              "pdflatex -synctex=1 -interaction=nonstopmode",
              "pdflatex -synctex=1 -interaction=nonstopmode"
          ]
      },

3. Theme

  • Adaptive.sublime-theme: This will change the sidebar to a dark colored background. Preference -> Themes -> Adaptive.sublime-theme
  • Monokai: Preference -> Color Schemes -> Monokai

3. SumatraPDF Settings - latex

  • win + R
  • Type cmd, press enter
  • Type sumatrapdf.exe -inverse-search "\"E:\Sublime3\sublime_text.exe\"%f:%l\", press enter

4. Shortcuut keys

  • ctrl + k, ctrl + b: 隐藏/显示侧边栏
  • ctrl + k, ctrl + 1: 折叠/显示所有代码
  • ctrl + b: 编译
  • ctrl + shift + b: 选择编译方式
